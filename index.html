Install

Install with npm

$ npm i -g gitfit

Install with Yarn

$ yarn global add gitfit

Run with npx (without installing)

$ npx gitfit

Usage

    Usage:

        $ gitfit  []

    Options:

        -h, --help              Display help information
        -v, --version           Output Initify version

    Examples:

        $ gitfit init
        $ gitfit start 
        $ gitfit finish
